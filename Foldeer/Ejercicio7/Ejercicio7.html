Versión Fecha de salida Cambios Importantes
0.3.0 29/05/13 Salida pública oficial

17.0.0 20/10/20 Añade una nueva transformacion JSX. Cambios en la delegación de eventos. No introduce nuevas "features" para permitir el update gradual de las versionse mas antiguas
18.0.0 29/03/22 React concurrente. "Batching" automático. Nuevas "Suspense Features", transiciones y apis de renderizado de cliente y servidor. Nuevos hooks. Nuevos comportamientos del modo estricto

Dato curioso

"React no es un framework"

De acuerdo con el sitio web oficial de React es una biblioteca de JavaScript para la construcción de interfaces de usuario, por lo que la respuesta corta es que React es una biblioteca de JavaScript, pero el hecho es que tiene varias características de un framework.
